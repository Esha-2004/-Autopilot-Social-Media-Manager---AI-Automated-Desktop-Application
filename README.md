# Autopilot-Social-Media-Manager-AI-Automated-Desktop-Application

A fully automated, multi-agent AI system designed to manage LinkedIn content for brand marketing. This project simulates an intelligent social media manager that operates independently—generating, reviewing, and scheduling high-quality posts without human intervention.

Built for EAI Systems, the system continuously analyzes trending AI news and transforms it into professional LinkedIn content aligned with brand voice and guidelines.

🎯 Project Objective

The objective of this project was to develop an autonomous AI-driven workflow that:

Scans trending AI/tech news daily

Generates professional LinkedIn posts
Ensures brand consistency through automated review
Schedules and stores approved content
Runs continuously for 5 days without manual input

🤖 System Architecture (Multi-Agent AI)
🔹 Manager Agent
Fetches and analyzes trending AI/tech news
Identifies top 3 relevant topics daily

✍️ Writer Agent
Generates 3 high-quality LinkedIn posts per day
Maintains a consistent, professional brand voice

🧠 Critic Agent
Reviews content based on brand guidelines
Rejects and triggers rewrites (up to 3 attempts)
Ensures quality and consistency

📅 Scheduler
Saves approved posts as .txt files
Simulates publishing via automation tools like Make.com / SocialChamp

⚙️ Key Features

✅ Fully autonomous multi-agent workflow

✅ AI-powered content generation using OpenRouter

✅ Built-in quality control with feedback loop

✅ Daily automation for 5 consecutive days

✅ Generates a total of 15 LinkedIn posts

✅ Scalable and adaptable for other brands
